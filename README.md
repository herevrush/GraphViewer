**GraphViewer**

This is a repository for GraphViewer Code Sample to plot graphs.

Try it

git clone https://github.com/herevrush/GraphViewer.git

cd GraphViewer

mvn install

java -jar target/GraphViewer-1.0-SNAPSHOT-jar-with-dependencies.jar

Prerequisites :-

Oracle Java 8

Maven
